# 🚀 React Learning Journey

## 📌 About This Repository

This repository contains my learning journey of **React.js**.
I have studied the fundamentals and core concepts of React and implemented small projects to understand how modern frontend development works.

React is a JavaScript library used to build fast and interactive user interfaces, especially for single-page applications (SPA).

---
